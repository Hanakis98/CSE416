1. Download and extract the project source.

2. In a shell, enter the ./backend/ folder.

3. Run "npm install".

4. Run "npm install nodemon".

5. Run "nodemon index.js". If on Windows, run "npx nodemon index.js".

6. Open a new shell.

7. Enter the ./frontend/ folder and run "npm install".

8. Run "npm start".

9. Open the localhost port in a browser. (Usually localhost:3000)
